## Technical Stack

- **Attribution Engine:** Markov-Shapley hybrid with Shapley value decomposition
- **Marketing Mix Modeling:** Bayesian hierarchical models with adstock and Hill saturation
- **Identity Resolution:** Probabilistic graph with S_ij scoring, Gaussian Mixture clustering
- **Real-Time Processing:** Apache Kafka, Apache Flink SQL, sub-100ms latency
- **Causal Inference:** DAG-based modeling, propensity scoring, geo-lift experiments
- **Incrementality:** Randomized holdout design, Difference-in-Differences, Bayesian Structural Time Series
