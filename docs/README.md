## Who This Is For

- **Data teams** at brands spending $1M+/month on media who need measurement infrastructure they can actually trust
- **Marketing scientists** implementing hybrid attribution (MMM + incrementality + identity resolution)
- **Analytics leaders** building causal inference capabilities for media optimization

## Technical Stack

- **Attribution Engine**: Markov-Shapley hybrid with Shapley value decomposition
- **Identity Resolution**: Probabilistic graph with S_ij scoring, mobile device fingerprinting
- **Real-Time Processing**: Kafka streams, Flink SQL, sub-500ms latency
- **MMM**: Bayesian hierarchical models with media carryover effects
- **Causal Inference**: DAGs, propensity scoring, geo-lift testing
